
Methods to optimize QA:
- Automate validation 
- profile data regularly to understand
- Design clear and Reusable data quality Rules
- Remove duplicate records
- Remove empty field column to make memory efficient
- use regex to simplify and easy 
- define range of values like price>0
- define date formats
- Profile each columns for missing values, duplicates and spelling mistakes
- Parallel or distributed processing for large data sets
- Track QA results and failures over time and do QA based on that 
- Do QA stage by stage
	> Initially do overall checking like accuracy and completeness of data.
	> Then go for detailed checking like values and checking urls
